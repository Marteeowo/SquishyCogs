# SquishyCogs
Just a smol collection of cogs for my personal use, but feel free to have a look!

# Installation
`[p]repo add squishycogs https://github.com/Quashera/SquishyCogs`

`[p]cog install squishycogs <cog-name>`

`[p] is your prefix`
